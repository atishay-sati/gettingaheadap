Arrays

Store and manipulate collections of data using arrays.

Arrays Overview

An array is a fixed-size collection of elements of the same type. Use arrays to store and manipulate data efficiently.

int[] numbers = new int[5];
numbers[0] = 10;
numbers[1] = 20;
System.out.println(numbers[0]); // Output: 10

Traversing Arrays

Use loops to access each element in an array.

for (int i = 0; i < numbers.length; i++) {
    System.out.println(numbers[i]);
}

Common Array Algorithms

Find the sum or maximum value in an array:

// Sum
int sum = 0;
for (int n : numbers) {
    sum += n;
}
// Max
int max = numbers[0];
for (int n : numbers) {
    if (n > max) max = n;
}
