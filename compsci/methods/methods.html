Methods & Parameters

Write reusable code with methods, parameters, and return values.

Defining Methods

Methods are reusable blocks of code. Define a method with a return type, name, and parameters.

public int add(int a, int b) {
    return a + b;
}

Calling Methods

Call a method by using its name and passing arguments.

int sum = add(3, 4); // sum is 7

Parameters and Return Values

Methods can take parameters and return values. Use void if no value is returned.

public void greet(String name) {
    System.out.println("Hello, " + name);
}

greet("Sam"); // Output: Hello, Sam
