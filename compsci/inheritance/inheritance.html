Inheritance

Extend classes and use polymorphism in Java.

Inheritance Basics

Inheritance allows a class to acquire properties and methods from another class. The extends keyword is used.

public class Animal {
    public void eat() {
        System.out.println("Animal eats");
    }
}

public class Dog extends Animal {
    public void bark() {
        System.out.println("Woof!");
    }
}

Dog d = new Dog();
d.eat(); // Output: Animal eats
d.bark(); // Output: Woof!

Polymorphism

Polymorphism lets you use a superclass reference for subclass objects.

Animal a = new Dog();
a.eat(); // Output: Animal eats
// a.bark(); // Error: bark() not in Animal

Method Overriding

Subclasses can override methods from the superclass.

public class Animal {
    public void speak() {
        System.out.println("Animal sound");
    }
}

public class Cat extends Animal {
    @Override
    public void speak() {
        System.out.println("Meow");
    }
}

Animal c = new Cat();
c.speak(); // Output: Meow
