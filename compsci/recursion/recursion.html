Recursion

Solve problems with recursive methods in Java.

What is Recursion?

Recursion is when a method calls itself to solve a smaller version of a problem. Each recursive call should move closer to a base case.

// Print numbers from n down to 1
public void countdown(int n) {
    if (n == 0) return; // base case
    System.out.println(n);
    countdown(n - 1); // recursive call
}

countdown(3); // Output: 3 2 1

Factorial Example

The factorial of n (n!) is the product of all positive integers up to n. Recursion is a natural fit for this problem.

public int factorial(int n) {
    if (n == 0) return 1; // base case
    return n * factorial(n - 1);
}

System.out.println(factorial(4)); // Output: 24

When to Use Recursion

  • When a problem can be broken into similar subproblems
  • When the problem has a clear base case
  • Examples: factorial, Fibonacci, searching trees
