Classes & Objects

Understand object-oriented programming basics in Java.

What is a Class?

A class is a blueprint for creating objects. It defines properties (fields) and behaviors (methods).

public class Dog {
    String name;
    int age;
    public void bark() {
        System.out.println("Woof!");
    }
}

Creating Objects

Objects are instances of classes. Use the new keyword to create an object.

Dog myDog = new Dog();
myDog.name = "Buddy";
myDog.age = 3;
myDog.bark(); // Output: Woof!

Constructors

Constructors initialize new objects. They have the same name as the class.

public class Dog {
    String name;
    int age;
    public Dog(String n, int a) {
        name = n;
        age = a;
    }
}
Dog d = new Dog("Max", 2);
