Variables & Data Types

Learn about primitive types, declaring variables, and type casting in Java.

Primitive Data Types

Java supports several primitive data types: int, double, boolean, char, etc. Variables are containers for storing data values.

int age = 16;
double price = 19.99;
boolean isStudent = true;
char grade = 'A';

Use the correct type for the data you want to store. For example, use int for whole numbers and double for decimals.

Declaring and Initializing Variables

Variables must be declared with a type before use. You can assign a value at the time of declaration or later.

int score;
score = 100;

Type Casting

Type casting converts a variable from one type to another. For example, converting a double to an int:

double d = 9.7;
int i = (int) d; // i is 9
