ArrayLists

Use dynamic lists for flexible data storage in Java.

ArrayList Basics

An ArrayList is a resizable array from Java's java.util package. It can grow or shrink as needed.

import java.util.ArrayList;
ArrayList<String> names = new ArrayList<>();
names.add("Alice");
names.add("Bob");
System.out.println(names.get(0)); // Output: Alice

Common ArrayList Methods

  • add(value) - Adds an element
  • get(index) - Gets element at index
  • set(index, value) - Sets element at index
  • remove(index) - Removes element at index
  • size() - Returns number of elements
names.set(1, "Charlie");
names.remove(0);
System.out.println(names.size()); // Output: 1

Traversing ArrayLists

Use loops to access each element in an ArrayList.

for (String name : names) {
    System.out.println(name);
}
